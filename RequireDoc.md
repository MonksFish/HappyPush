# 2需求简介

1.  **个人中心**

    a.  **用户注册**

```markdown
i.  用户昵称,默认使用用户微信昵称

ii. 用户地区，默认使用用户微信地区

iii. 生成用户唯一标识
```
​       b.**关系绑定**

```markdown
i.  目标关系定义

ii. 绑定机制-发起邀请绑定

         1.  搜索目标用户【可通过（用户ID）】

         2.  搜索机制风控

             a.  一个用户只能对同一个用户发起一次邀请

             b.  一个用户一天内只能发起【定制次数】次邀请（同如果要没有上限，赞赏我们等渠道获取发起次数）

iii. 绑定历史记录

        1.  内含历史邀请记录，邀请状态，未邀请成功呈红色状态
```

2. **推送管理**

   a.  **推送模版类型自定义**

   ```markdown
   i.  分基础化模版 & 定制化模版
   
       1.  每日热点
   
       2.  暖心文学
   
       3.  励志鸡汤
   
       4.  当地疫情状态
   
       5.  唯美爱情
   
   ii. 模版目标推送时间管理
   
   iii. 推送目标绑定管理
   ```

3. 后台管理中心

   ```markdown
   a.  用户管理
   ```

4.  联系我们-加入我们-打赏我们

# 3总体需求概述 

前提用户进行注册，进行拟定推送内容【可定制】后，关联绑定推送目标【可拟定目标关系】后，设置目标模版发送时期【模版目标管理】，达到目标化公众号推送
